Build a Plugin

1

Create the directory

plugins/my-plugin/
├── plugin.json
└── skills/
    └── my-skill/
        └── SKILL.md
2

Write the manifest

{
  "name": "my-plugin",
  "description": "What it does",
  "version": "1.0.0",
  "skills": "./skills"
}
3

Register it

{
  "name": "Palmshed Official",
  "plugins": [
    {
      "name": "my-plugin",
      "source": "./plugins/my-plugin"
    }
  ]
}

Plugin Structure

📜

Skills

Drop a SKILL.md in skills/<name>/. YAML frontmatter + markdown body.

⚡

Commands

Add any .md file in commands/. Filename becomes the command name.

🤖

Agents

Add any .md file in agents/. Filename becomes the agent name.

🔌

MCP Servers

Add .mcp.json at the plugin root with your server config.
